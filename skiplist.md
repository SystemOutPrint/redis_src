# 跳跃表

## 0x01 定义
```c
typedef struct zskiplist {
    struct zskiplistNode *header, *tail; // 跳跃表的头结点和尾节点
    unsigned long length; // 跳跃表的长度
    int level; // 跳跃表的深度
} zskiplist;
```
```c
typedef struct zskiplistNode {
    robj *obj; // 当前节点的值对象
    double score; // 当前节点的评分
    struct zskiplistNode *backward; // 当前level的前驱节点
    struct zskiplistLevel {
        struct zskiplistNode *forward; // 当前level的后继节点
        unsigned int span; // 跨度
    } level[]; // 层
} zskiplistNode;
```

## 0x03 跳跃表搜索原理

### 查找分值为6.0的元素
1、选取最高层的头结点(0.0, L4)<br/>
2、得到(0.0, L4)的后继节点(7.0, L4)<br/>
3、判断分值6.0 < 7.0<br/>
4、回到(0.0, L4)<br/>
5、选择相同分值的下一层节点(0.0, L3)<br/>
6、得到(0.0, L3)的后继节点(4.0, L3)<br/>
7、判断分值6.0 > 4.0<br/>
8、选择(4.0, L3)<br/>
9、得到(4.0, L3)的后继节点(7.0, L3)<br/>
10、判断分值6.0 < 7.0<br/>
11、回到(4.0, L3)
12、选择相同分值的下一层节点(4.0, L2)<br/>
13、得到(4.0, L2)的后继节点(7.0, L2)<br/>
14、判断6.0 < 7.0<br/>
15、回到(4.0, L2)
16、选择相同分值的下一层节点(4.0, L1)<br/>
17、得到(4.0, L1)的后继节点(6.0, L1)<br/>
18、判断6.0 == 6.0<br/>
19、完成

## 0x04 跳跃表插入原理

### 层数生成
原理同抛硬币，生成n层节点的概率为(1/2)^n。

### 查找新节点的前驱和后继
将每一层都视为一个链表，在这个链表中寻找到相应的位置，记录下这个位置的前驱和后继。

### 插入节点
将节点插入到每层的链表中。
